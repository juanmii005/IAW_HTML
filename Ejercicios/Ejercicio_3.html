HTML (Lenguaje de Marcas de Hipertexto, del inglés HyperText Markup Language ) es el componente más básico de la Web. Definir el significado y la estructura del contenido web. Además de HTML, generalmente se utilizan otras tecnologías para describir la apariencia/presentación de una página web ( CSS ) o la funcionalidad/comportamiento ( JavaScript ).

Información sobre IAW

Una aplicación web es una herramienta o software que se ejecuta en el navegador para realizar una serie de tareas online. Hay muchos tipos distintos, desde herramientas de cálculo, aplicaciones de redes sociales o de correo electrónico hasta sistemas complejos para gestionar contenidos. Cada uno está diseñado para cumplir una función específica dependiendo de las necesidades del usuario.
